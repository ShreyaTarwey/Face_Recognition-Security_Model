About The Project:

The project is based on the problem statement: 'Facial Recognition System for Security Purposes'. The system uses biometrics to map facial features from a photograph or video. It compares the information with a database of known faces to find a match.

This system is highly secure as unlike passwords where someone/something just has to leak a malware into a device and steal hundreds of passwords at once, facial recognizition is composed of biometrics that makes it hard to hack.

This project has made use of an API namely 'face-api.js' to achieve the final product. The file loads three important modules namely 'Face Recognition Net', 'Face Landmark 68 Net' and 'SSD Mobile Net V1' from the API call and uses these to identify the faces that is pre-fed to the system.

Key Features

Listed below are the significant features of this website:

Facial Recognition

Under this feature, you can use the facial recognition module to recognize a person's face via the use of a webcam. After clicking upon it, make sure to give the 'Allow Access' permission to the camera to utilize the feature.


Video

Under this feature, you can identify the faces of person/persons in a particular video frame. The faces of the people to be recognized is pre-loaded into into the system. By making use of biometric facial feature similarity, the API deduces if they are the same person or not.
